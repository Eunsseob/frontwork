객체

* 객체의 선언 및 사용

객체는 중괄호{}를 사용하여 생성하고,
  중괄호 안에 이 객체를 필요한 속성을 정의함.
속성명.속성값의 형태로 정의(key:value)
  속성값으로는 모든 자료형의 값을 담을 수 있다. 

        변수 = {
            속성명:속성값,
            속성명:속성값,
            속성명:속성값,
            ...
        }

* 객체에서의 반복문

객체가 가지고 있는 모든 속성들에 순차적으로 접근하고자 한다면 반복문 사용 가능 단, 단순한 for loop로는 불가, 향상된 for문만 사용 가능
